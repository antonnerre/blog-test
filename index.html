From data queries to predictions

We will start from scratch and cover different aspects of a data science project: statistical inference, data visualization and exploration, machine learning, networks, etc.

In this serie of 6 notebooks, we will cover how it is possible to collect few different data and try to get insights from them. My love for music naturally led me to gather data from Spotify, using the very convenient ``spotipy`` package. We will cover topics such as data manipulation, exploratory data analysis, statistical inference, networks and predictions with machine learning methods (gradient boosting and neural networks).

It is possible to gather very interesting data at the song level, but that might be for another serie. For this one, remember that the only data we will have access to are:

  • the artists' or bands' names,
  • the genre(s) they belong to,
  • their popularity on Spotify,
  • and their number of followers.

Let us see how far we can go with that!
